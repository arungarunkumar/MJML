MJML Mailjet Markup Language

What is MJML ?

MJML is a resonsive email framework.
It's used to buid a responsive email.

Notes: It was built in ReactJS, its takes the MJML
as an input and outputs as HTML

and
Installation

Requirements

Node >= 4.2.x

Installation

npm install -g mjml

Install MJML globally

Getting Started
Step by Step

Step 1:
Create the file with .mjml extensions for example (index.mjml)

Step 2:
Code your mailer in MJML file

Step 3:
After complete your coding compile the MJML file to get the HTML file

Step 4:
Once we get the generated HTML we can send it with an email service provider.

Commands to compile the MJML file

Compile the MJML file and output the result in .html
mjml -r input.mjml
Redirect the result to a file
mjml -r input.mjml -o output.html
Watch a file and compile every time the file changes
mjml -w input.mjml -o output.html

How to make a responsive Grid

By using "mj-column" and "mj-group" to make a responsive grid

Diffrence between "mj-column" and "mj-group"

mj-column
If we have four columns in desktop in mobile it will render as 1 column

mj-group
If we have four columns in desktop in mobile it will render as 2 column per row

Tips and Tricks

  • To avoid repeated or common style for each section we can add class in components and style it inside "mj-attributes" tag

  • Inside HTML tag mustn't use any mjml tags but, inside mjml tag we can use HTML tags.

  • To give links to images and text

  • We can add extenal MJML file by using "mj-include" to add in our email template 

     <mj-include path="path/file_name" />
